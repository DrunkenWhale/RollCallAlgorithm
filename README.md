我必须得承认 这题目很离谱  
任何算法都不可能高效的找到随机值  
随机缺课的学生无法有效的被找出
根据

$$ E = \frac{五门课程的有效点名次数}{总请求次数} $$

我们知道 在有强相关性特征的情况下  
__点名次数越少 E越高__

最优方案必定需要全点到固定翘课的学生  
实现`E`值最高且点名范围最广
